# Шпаргалка: Списки в Python

## Что такое список
Список (`list`) — **упорядоченная изменяемая коллекция**. Можно добавлять, удалять и изменять элементы.

```python
fruits = ['яблоко', 'банан', 'вишня']
empty = []  # Пустой список
```

## Создание списка

```python
# Из кода
numbers = [1, 2, 3, 4, 5]

# Из строки
chars = list('hello')           # ['h', 'e', 'l', 'l', 'o']

# Из range
nums = list(range(5))           # [0, 1, 2, 3, 4]

# Из кортежа
items = list(('a', 'b', 'c'))   # ['a', 'b', 'c']
```

## Доступ к элементам

```python
animals = ['ёж', 'кит', 'слон', 'питон']

print(animals[0])    # ёж (первый)
print(animals[-1])   # питон (последний)

# Срезы
print(animals[1:3])  # ['кит', 'слон']
print(animals[::-1]) # ['питон', 'слон', 'кит', 'ёж'] (разворот)
```

## Основные операции

```python
# Длина
len(animals)         # 4

# Проверка вхождения
'кит' in animals     # True

# Конкатенация
[1, 2] + [3, 4]      # [1, 2, 3, 4]

# Повторение
[1, 2] * 3           # [1, 2, 1, 2, 1, 2]
```

## Методы списков

### Добавление

```python
shopping = ['молоко', 'хлеб']

# Добавить в конец
shopping.append('яйца')           # ['молоко', 'хлеб', 'яйца']

# Добавить несколько элементов
shopping.extend(['масло', 'сыр']) # ['молоко', 'хлеб', 'яйца', 'масло', 'сыр']

# Вставить на позицию
shopping.insert(1, 'батон')       # ['молоко', 'батон', 'хлеб', ...]
```

### Изменение

```python
shopping[0] = 'кефир'    # замена элемента
```

### Удаление

```python
# По индексу (del)
del shopping[0]           # удалить первый элемент
del shopping[1:3]         # удалить срез

# По индексу с возвратом (.pop())
item = shopping.pop()     # удалить и вернуть последний
item = shopping.pop(0)    # удалить и вернуть первый

# По значению (.remove())
shopping.remove('хлеб')   # удалить первое вхождение

# Очистить всё
shopping.clear()
```

### Поиск и подсчёт

```python
pizza = ['пепперони', 'пепперони', 'барбекю']

# Количество вхождений
pizza.count('пепперони')   # 2

# Индекс первого вхождения
pizza.index('барбекю')     # 2
```

### Сортировка и разворот

```python
nums = [3, 1, 4, 1, 5]

# Сортировка на месте
nums.sort()                 # [1, 1, 3, 4, 5]
nums.sort(reverse=True)     # [5, 4, 3, 1, 1]

# Новая отсортированная копия
sorted_nums = sorted(nums)

# Разворот на месте
nums.reverse()              # [5, 4, 1, 1, 3]

# Новый развёрнутый список
reversed_nums = list(reversed(nums))
# Или срезом: nums[::-1]
```

## Обход списка

```python
items = ['a', 'b', 'c']

# По элементам
for item in items:
    print(item)

# По индексам
for i in range(len(items)):
    print(i, items[i])

# По индексам и элементам
for idx, val in enumerate(items):
    print(idx, val)
```

## Копирование списков

```python
original = [1, 2, 3]

# Три способа поверхностного копирования
clone1 = original[:]      # срез
clone2 = original.copy()  # метод copy()
clone3 = list(original)   # конструктор

# Глубокое копирование (для вложенных списков)
import copy
full_clone = copy.deepcopy(original)
```

## Важные отличия

### `==` vs `is`

```python
a = [1, 2]
b = [1, 2]
c = a

print(a == b)   # True (одинаковые значения)
print(a is b)   # False (разные объекты в памяти)
print(a is c)   # True (один и тот же объект)
```

### `+` vs `+=`

```python
a = [1, 2]
b = a

a = a + [3, 4]   # создаётся новый список
print(b)         # [1, 2] (старый не изменился)

a = [1, 2]
b = a
a += [3, 4]      # изменяется исходный список
print(b)         # [1, 2, 3, 4] (b тоже изменился!)
```

## Распаковка списков

```python
nums = [1, 2, 3, 4, 5]

# Простая распаковка
one, two, three = [1, 2, 3]

# Со звездочкой (собирает остаток)
first, *middle, last = nums
print(first)   # 1
print(middle)  # [2, 3, 4]
print(last)    # 5

# Распаковка в print
print(*nums)   # 1 2 3 4 5
```

## Полезные функции

```python
nums = [10, 30, 20]

max(nums)      # 30
min(nums)      # 10
sum(nums)      # 60
len(nums)      # 3
```

## Сортировка пузырьком (для понимания)

```python
nums = [5, 3, 8, 1, 2]
n = len(nums)

for i in range(n):
    for j in range(0, n - i - 1):
        if nums[j] > nums[j + 1]:
            nums[j], nums[j + 1] = nums[j + 1], nums[j]

print(nums)  # [1, 2, 3, 5, 8]
```

## Главное запомнить

| Что можно? | А можно! |
|------------|----------|
| Изменять элементы | `lst[0] = 'новое'` |
| Добавлять | `.append()`, `.extend()`, `.insert()` |
| Удалять | `.pop()`, `.remove()`, `del` |
| Сортировать | `.sort()`, `sorted()` |
| Копировать | `[:]`, `.copy()`, `list()` |

> **P. S.** Список — это швейцарский нож в Python. Используйте его везде, где нужна изменяемая упорядоченная коллекция!
