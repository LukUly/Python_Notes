# Шпаргалка: Функции с переменным числом аргументов

## Что это?
Функции, которые могут принимать **любое количество аргументов**.

```python
print(1)                # 1 аргумент
print(1, 2, 3, 4, 5)    # 5 аргументов
print("a", [1,2], None) # разные типы
```

---

## *args — позиционные аргументы

**Звёздочка** `*` перед параметром собирает все лишние позиционные аргументы в **кортеж**.

```python
def super_greet(*names):
    for name in names:
        print(f"Привет, {name}!")

super_greet("Вася")              # names = ("Вася",)
super_greet("Вася", "Петя", "Маша")  # names = ("Вася", "Петя", "Маша")
```

### С обычными параметрами

```python
def party_greet(host, *guests):
    print(f"Ведущий: {host}")
    for guest in guests:
        print(f"Гость: {guest}")

party_greet("Джон", "Дейенерис", "Тирион")
# host = "Джон"
# guests = ("Дейенерис", "Тирион")
```

### Обработка пустого списка

```python
def lonely_greet(*names):
    if names:
        for name in names:
            print(f"Привет, {name}!")
    else:
        print("Никого нет...")
```

## Значения по умолчанию

Делают параметры **необязательными**.

```python
def make_burger(type_of_meat, with_onion=False, with_tomato=True):
    print(f"Бургер с {type_of_meat}")
    if with_onion:
        print("  + лук")
    if with_tomato:
        print("  + помидор")

# Разные способы вызова
make_burger("говядина")                      # по умолчанию
make_burger("курица", with_onion=True)       # только лук
make_burger("свинина", with_onion=True, with_tomato=False)
```

**Важно:** параметры со значениями по умолчанию идут **после** обязательных.

```python
# ✅ Правильно
def func(a, b, c=10, d=20): pass

# ❌ Неправильно
# def func(a, c=10, b, d=20): pass  # SyntaxError!
```

## **kwargs — именованные аргументы

**Две звёздочки** `**` собирают все лишние именованные аргументы в **словарь**.

```python
def create_profile(name, age, **extra_info):
    print(f"Имя: {name}")
    print(f"Возраст: {age}")
    for key, value in extra_info.items():
        print(f"{key}: {value}")

# Вызов
create_profile("Вася", 25)
create_profile("Петя", 30, любимый_цвет="зелёный", iq=300, хобби="шахматы")
```

## Все вместе: полный порядок параметров

**Правильный порядок:**
1. Обычные (позиционные) параметры
2. `*args` (если есть)
3. Параметры со значениями по умолчанию
4. `**kwargs` (если есть)

```python
def ultimate_function(
    required_arg,           # обязательный
    *args,                  # лишние позиционные → кортеж
    default_arg="по умолчанию",  # со значением по умолчанию
    **kwargs                # лишние именованные → словарь
):
    print(f"Обязательный: {required_arg}")
    print(f"*args: {args}")
    print(f"default_arg: {default_arg}")
    print(f"**kwargs: {kwargs}")

# Пример вызова
ultimate_function(
    "первый",           # required_arg
    "это", "в", "args", # уйдёт в *args
    default_arg="я новый",
    custom="попадает в kwargs",
    another=42
)
```

## Распаковка при вызове

### Распаковка списка/кортежа → позиционные аргументы (`*`)

```python
def greet(a, b, c):
    print(a, b, c)

my_list = [1, 2, 3]
greet(*my_list)        # → greet(1, 2, 3)

my_tuple = (10, 20, 30)
greet(*my_tuple)       # → greet(10, 20, 30)

# Можно несколько
greet(*[1, 2], *[3, 4])  # если функция принимает 4 аргумента
```

### Распаковка словаря → именованные аргументы (`**`)

```python
def make_burger(meat, with_onion=False, with_tomato=True):
    print(f"Бургер с {meat}, лук={with_onion}, помидор={with_tomato}")

my_order = {
    "meat": "говядина",
    "with_onion": True,
    "with_tomato": False
}
make_burger(**my_order)  # → make_burger(meat="говядина", with_onion=True, with_tomato=False)
```

### Комбинирование

```python
def func(a, b, c, d):
    print(a, b, c, d)

func(1, *[2, 3], d=4)           # 1, 2, 3, 4
func(*[1, 2], **{"c": 3, "d": 4})  # 1, 2, 3, 4
```

## Важные ловушки

### ❌ Изменяемый тип как значение по умолчанию

```python
# НЕ ДЕЛАЙТЕ ТАК!
def bad_append(value, bucket=[]):
    bucket.append(value)
    return bucket

print(bad_append(1))  # [1]
print(bad_append(2))  # [1, 2] ← сюрприз!
```

### ✅ Правильный способ

```python
def good_append(value, bucket=None):
    if bucket is None:
        bucket = []
    bucket.append(value)
    return bucket

print(good_append(1))  # [1]
print(good_append(2))  # [2]
```

## pass и многоточие (...)

```python

def calculate_something():
    pass

# Способ 2: ... (многоточие)
def future_function():
    ...

# Оба варианта рабочие
```

## Смотрим сигнатуру функции

```python
import inspect

def my_func(a, b=10, *args, c=20, **kwargs):
    pass

print(inspect.signature(my_func))
# (a, b=10, *args, c=20, **kwargs)
```

## Краткая памятка

| Конструкция | Что делает | Где используется |
|-------------|------------|------------------|
| `*args` в параметрах | Собирает лишние позиционные аргументы в **кортеж** | В определении функции |
| `**kwargs` в параметрах | Собирает лишние именованные аргументы в **словарь** | В определении функции |
| `*список` при вызове | Распаковывает список в позиционные аргументы | При вызове функции |
| `**словарь` при вызове | Распаковывает словарь в именованные аргументы | При вызове функции |
| `param=значение` | Значение по умолчанию для параметра | В определении функции |

---

## Именование по договорённости

- `*args` — от **arguments** (позиционные)
- `**kwargs` — от **keyword arguments** (именованные)

Но можно использовать и другие имена, особенно когда понятно, что там лежит:

```python
def super_greet(*names):     # вместо *args
    ...

def create_user(**user_data): # вместо **kwargs
    ...
```

> **P. S.** Хотите гибкую функцию? Добавьте `*args` и `**kwargs`. Но не переусердствуйте — код должен быть понятным!
