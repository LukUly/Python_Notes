# Функции высшего порядка и lambda

## Что это?
Функции в Python — **объекты первого класса**. Их можно:
- присваивать переменным
- хранить в коллекциях
- передавать в другие функции
- возвращать из функций

---

## Функция как объект

```python
def greet(name):
    return f"Привет, {name}!"

# Присваивание
hello = greet
print(hello("Вася"))  # Привет, Вася!

```

---

## Функция как аргумент (функции высшего порядка)
Если функция принимает другую функцию в качестве аргумента - она называется функцией высшего порядка

```python
def apply_twice(func, x):
    return func(func(x))

def inc(n):
    return n + 1

print(apply_twice(inc, 10))  # 12
```

## map() — преобразование элементов

```python
map(функция_преобразования, коллекция)
```

```python
# Удвоить каждый элемент
def double(x):
    return x * 2

doubled = list(map(double, [1, 2, 3]))
print(doubled)  # [2, 4, 6]

# Перевести температуры в Фаренгейты
def to_fahrenheit(c):
    return c * 9/5 + 32

celsius = [0, 20, 30, 100]
fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit)  # [32.0, 68.0, 86.0, 212.0]

# С несколькими коллекциями
def add(a, b):
    return a + b

list1 = [1, 2, 3]
list2 = [10, 20, 30]
sums = list(map(add, list1, list2))
print(sums)  # [11, 22, 33]
```

## sorted() и .sort() с key

```python
sorted(коллекция, key=функция, reverse=False)
# .sort() — то же, но изменяет список на месте
```

```python
words = ["кот", "грабли", "вафля", "дракоша"]

# По длине слова
def length(w):
    return len(w)

sorted_words = sorted(words, key=length)
print(sorted_words)  # ['кот', 'вафля', 'грабли', 'дракоша']

# По последней букве
def last_letter(w):
    return w[-1]

by_last_letter = sorted(words, key=last_letter)
print(by_last_letter)  # ['вафля', 'грабли', 'кот', 'дракоша']

# Сложные объекты
students = [
    {"name": "Алиса", "age": 25, "score": 80},
    {"name": "Боб", "age": 30, "score": 90},
    {"name": "Вася", "age": 20, "score": 75}
]

# По возрасту
def student_age(s):
    return s["age"]

by_age = sorted(students, key=student_age)

# По убыванию баллов
def student_score(s):
    return s["score"]

by_score_desc = sorted(students, key=student_score, reverse=True)

# По нескольким критериям (кортеж)
def multi_key(s):
    return (s["age"], -s["score"])

multi = sorted(students, key=multi_key)
```

## min() и max() с key

```python
min(коллекция, key=функция)  # элемент с минимальным значением key
max(коллекция, key=функция)  # элемент с максимальным значением key
```

```python
words = ["кот", "грабли", "вафля", "дракоша"]

def length(w):
    return len(w)

longest = max(words, key=length)      # 'дракоша'
shortest = min(words, key=length)     # 'кот'

# По последней букве (максимальная буква в конце)
def last_letter(w):
    return w[-1]

last_max = max(words, key=last_letter)  # 'дракоша' (я > а, б, л)

# Со словарями
products = [
    {"name": "печенье", "price": 100, "discount": 0.1},
    {"name": "шоколад", "price": 95,  "discount": 0},
    {"name": "пирожное", "price": 120, "discount": 0.25}
]

# Самый дешёвый с учётом скидки
def final_price(p):
    return p["price"] * (1 - p["discount"])

cheapest = min(products, key=final_price)
print(cheapest["name"])  # пирожное
```

## Цепочки filter → map → sorted

```python
students = [
    {"name": "Алиса", "grades": [85, 92, 78], "age": 20},
    {"name": "Боб", "grades": [45, 60, 55], "age": 22},
    {"name": "Вася", "grades": [92, 95, 88], "age": 19},
]

# Вспомогательные функции
def add_average(s):
    avg = sum(s["grades"]) / len(s["grades"])
    return {**s, "avg": avg}

def is_young(s):
    return s["age"] < 21

def avg_score(s):
    return s["avg"]

# 1. Добавим средний балл
with_mean = map(add_average, students)

# 2. Фильтр по возрасту (< 21)
young = filter(is_young, with_mean)

# 3. Сортировка по среднему баллу (убывание)
top_young = sorted(young, key=avg_score, reverse=True)

# Результат
for student in top_young:
    print(f"{student['name']}: {student['avg']:.1f}")
```

```python
def add_average(s):
    return {**s, "avg": sum(s["grades"])/len(s["grades"])}

def is_young(s):
    return s["age"] < 21

def avg_score(s):
    return s["avg"]

result = sorted(
    filter(is_young,
        map(add_average, students)
    ),
    key=avg_score, reverse=True
)
```

## Функция, возвращающая функцию (замыкание)

```python
def make_multiplier(k):
    def mul(x):
        return x * k
    return mul

double = make_multiplier(2)
triple = make_multiplier(3)

print(double(10))  # 20
print(triple(10))  # 30
```

## Краткая памятка

| Задача | Решение |
|--------|---------|
| Применить функцию к каждому элементу | `map(func, coll)` |
| Отфильтровать элементы | `filter(predicate, coll)` |
| Отсортировать по ключу | `sorted(coll, key=func)` |
| Найти min/max по ключу | `min(coll, key=func)` |
| Вернуть функцию, которая помнит переменные | замыкание (def внутри def) |
